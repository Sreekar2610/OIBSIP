# Task 3: ATM Interface

This project simulates a basic ATM interface using Java.  
Users can log in with their credentials and perform several banking operations through the console.

## Features
- User authentication (PIN-based).
- Transaction history.
- Withdraw, deposit, and transfer functions.
- Balance inquiry.
- Secure and user-friendly menu-driven interface.
