# Task 4: Online Examination System

This project implements a console-based Online Examination System in Java.  
It provides students with a timed test environment, multiple-choice questions, and instant result calculation.

## Features
- User login authentication.
- Multiple-choice questionnaire.
- Timed exam session.
- Automatic scoring and result display.
- Option to review and retake the exam.
