# Task 5: Digital Library Management System

This project creates a simple Digital Library Management System using Java.  
The application allows users to manage book records, search for books, and perform basic library member operations via the console.

## Features
- Add, remove, and search for books.
- Track book availability and manage lending.
- User/member registration.
- View book and member lists.
- Console-based menu system for easy navigation.
